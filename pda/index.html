Push Down Automata Construction

Construct a PDA from the example calculator language (CL)

Syntax:

read input:
read (var)

assignment:
(var) := (val or expression)

while:
while (condition) { (body) } NOTE: no parens around condition, so chaining together expressions are not yet supported

if:
if (condition) { (body) } NOTE: Unlike while loops, this uses parens around the expression (but chaining expressions is still not supported)

write output:
write (var)

arithmetic:
(id) := (id or literal) (operation) (id or literal)

arithmetic operations supported: + - * /

boolean operations supported: == != > < <= >=
